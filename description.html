Описание XOs

Стартовая страница

На стартовой странице вы можете начать игру или посмотреть рекорды. Для старта как сервер нужно ввести только имя, для старта как клиент нужно ввести ip клиента

Выбор паараметров

Выбирается 3 параметра: высота, ширина в клетках игрового поля и число клеток, оторые нужно закрасить одним маркером в линию, для победы.

Игра

Каждый игрок внизу видит, чей сейчас ход. Для закраски клетки нужно ткнуть по ней мышкой.

Окончание

В любой момент игру можно покинуть, оппонент увдит, что игра покинута, поражение засчитано не будет.
